# Audio assets

Drop the ambient soundtrack here as `ambient-praefatio.mp3`.

**Good fits for the brand** (creative-commons / royalty-free):
- Free Music Archive: search "ambient piano" or "library" - https://freemusicarchive.org/
- Pixabay Music (free, no attribution required): https://pixabay.com/music/search/ambient/
- ccmixter.org for atmospheric tracks
- Search "candlelit library ambience" on YouTube Audio Library

**Specifications for the file:**
- Format: MP3, 128kbps (small file, good enough quality)
- Duration: 2-5 minutes (will loop)
- Volume normalised to -14 LUFS or quieter (we play at 18% in JS)
- Faded in/out at start/end so the loop point is invisible

Once added, push and it'll work automatically via the speaker icon top-right.
