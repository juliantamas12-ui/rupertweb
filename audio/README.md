## To replace the track

Drop a new file as `ambient-praefatio.mp3` here:

- Format: MP3, 128kbps
- Duration: 2-5 minutes (will loop)
- Volume normalised to -14 LUFS or quieter (we play at 18% in JS)
- Faded in/out at start/end so the loop point is invisible
